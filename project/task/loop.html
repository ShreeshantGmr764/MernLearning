loops types

  1. For loop
    for (initial expression, condition, increment/ decrement ){
    code to be executed
    }
    Example:
    for (let i = 0; i < 5; i++) {
    console.log(i);
    }
  2. While loop while (condition) {
    code to be executed increment/ decrement
    }
  3. Do while loop do {
    code to be executed
    } while (condition);
